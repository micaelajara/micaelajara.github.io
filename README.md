# micaelajara.github.io

Portfolio personal de **Micaela Jara** — QA Engineer · Tech Enthusiast · Project Maker.

🔗 **Live:** [micaelajara.github.io](https://micaelajara.github.io)

---

## Stack

HTML · CSS · JavaScript vanilla — sin frameworks, sin dependencias.

## Secciones

| Sección | Descripción |
|---|---|
| Hero | Badges de disponibilidad, idioma y años de experiencia auto-calculados |
| Sobre mí | Bio, stats animados (años QA / recomendaciones / dominios), card "Ahora mismo" |
| Cómo trabajo | Manifiesto en 4 cards numeradas |
| Proyectos | Carrusel: Penske (5 dominios), Freelance QA, Club Classics, Radio La Tribu |
| Stack | Herramientas actuales + en aprendizaje |
| Formación | Carrera universitaria + certificaciones |
| Intereses | Flip cards 3×2, hover en desktop / tap en mobile |
| Conocimientos | Accordion con búsqueda y filtros por categoría |
| Testimonios | Carrusel con 11 recomendaciones de LinkedIn |
| Contacto | Formulario (Formspree) + links |

## Features

- Bilingüe ES/EN con detección automática del idioma del dispositivo
- Preferencia guardada en `localStorage`
- Animaciones de scroll con `IntersectionObserver`
- Cursor personalizado salmon (desktop)
- Easter egg: click en Snoopy 🐾
- Contadores animados con easing cúbico
- Google Analytics (GA4)
- Open Graph para previews en LinkedIn / WhatsApp
- Responsive mobile-first
